# Usage

--8<-- "snippets/wip.md"

- Subscribe to mods on the steam workshop, install mods locally in `Documents/my games/Stationeers/mods`, or install mods from repos using the [`slp repos`](commands.md#repos) and [`slp repomods`](commands.md#repomods) console commands.
- Go to the Settings page on the main menu to adjust StationeersLaunchPad settings
- Click the SLP loading box on the bottom at startup (or disable the `AutoLoad` setting) to enable/disable/reorder mods before loading.
- Report issues and get help in the [Stationeers Discord](https://discord.gg/stationeers) #modding channel, or in the [Stationeers Modding Discord](https://discord.gg/5qZbPVTw2U)
    - Include a debug package generated with the `slp debugpkg` console command to aid in debugging
